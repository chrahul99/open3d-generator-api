# Contributing

Contributions are welcome.

Good first areas:

- Add a tested adapter config for a new open-source image-to-3D model.
- Improve Hunyuan3D setup docs for specific hardware.
- Add a small web UI for upload, preview, and download.
- Add artifact storage backends such as S3-compatible object storage.
- Add a durable queue for production deployments.

Please keep model weights out of this repository. Link to upstream model instructions instead.
